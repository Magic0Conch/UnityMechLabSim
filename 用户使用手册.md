# 电火花加工虚拟仿真实验系统 — 用户使用手册

本文档面向**教师与学生**，说明如何安装客户端、连接学校服务器并完成实验与考试。服务器部署与运维请参阅《部署手册.md》。

---

## 一、使用前准备

### 1.1 您需要具备

| 角色 | 需要 |
|------|------|
| 学生 / 教师 | 已安装好的 Unity 客户端（或教师提供的发布版 exe） |
| 全体用户 | 学校提供的**服务器地址**（由管理员下发） |

### 1.2 服务器地址格式

登录页「服务器地址」须填写完整 API 地址，格式为：

```
http://<服务器IP>:8899/api/values
```

示例：`http://49.233.160.65:8899/api/values`（以教师或管理员下发的 IP 为准）。

**注意**：地址末尾必须包含 `/api/values`。输入后点击确认，系统会自动保存，下次启动无需重复填写。

### 1.3 验证服务器是否可用

在浏览器中访问（将 IP 换成实际地址）：

- 健康检查：`http://49.233.160.65:8899/health` → 应显示 `{"status":"ok"}`
- 若无法打开，请联系实验课教师或机房管理员，不要自行修改端口。

## 二、登录与账号

### 2.1 启动客户端

1. 运行 Unity 工程发布版，或在 Unity 编辑器中打开场景 `Assets/Scenes/newScene/登录.unity` 后 Play。
2. 在登录页填写**服务器地址**、**用户名**、**密码**。
3. 点击登录。

### 2.2 账号说明

账号由**教师或管理员**创建与分发，首次使用请以教师通知为准。演示环境可能提供如下测试账号（正式环境务必修改密码）：

| 用户名 | 典型角色 | 登录后场景 |
|--------|----------|------------|
| teacher | 教师 | Tscene（管理端） |
| student1、student2… | 学生 | MainScene（实验流程） |

### 2.3 离线模式

登录页可选择「离线」进入仿真场景，仅可进行电火花虚拟操作练习，**无法**登录、上传作业、参加考试或同步成绩。正常实验课请使用联网登录。

## 三、学生端操作指南

学生登录后进入 **MainScene**，左侧为实验任务树，请按课程要求依次完成。

### 3.1 实验任务一览

| 序号 | 任务名称 | 主要内容 | 完成要求 |
|------|----------|----------|----------|
| 0 | 实验目的、内容 | 下载压缩包 | 下载「实验目的内容」 |
| 1 | 设备、仪器 | 阅读 PDF | 阅读设备仪器说明 |
| 2 | 阅读实验指导书 | 阅读 PDF | 阅读实验指导书 |
| 3 | 视频资源 | 获取视频链接 | 从服务器获取 videos |
| 4 | 安全操作考试 | 在线答题 | **得分 ≥ 90** |
| 5 | 职业工程素养测试 | 在线答题 | **得分 ≥ 90** |
| 6 | 人员分工 | 上传文件 | 上传分工文档 |
| 7 | 模型分配 | 上传文件 | 上传模型分配表 |
| 8 | 加工工艺安排 | 上传文件 | 上传工艺安排 |
| 9 | 凸凹模虚拟加工 | 上传文件 | 上传虚拟加工记录 |
| 10 | 凸凹模现实加工 | 上传文件 | 上传现实加工记录 |
| 11 | 电火花放电虚拟加工 | 软件内实验 | 在仿真场景中完成加工 |
| 12 | 电火花放电现实加工 | 上传文件 | 上传现实加工记录 |
| 13 | 尺寸检验 | 上传文件 | 上传尺寸数据 |
| 14 | 粗糙度检验 | 上传文件 | 上传粗糙度数据 |
| 15 | 实验总结 | 上传文件 | 上传总结报告 |

### 3.2 阅读 PDF

进入任务 1、2 后点击「开始任务」，系统内嵌 PDF 阅读器会打开对应文档。关闭阅读器后，阅读时长会记录并同步到服务器。

### 3.3 在线考试

1. 进入「安全操作考试」或「职业工程素养测试」。
2. 若提示未获取题库，点击 **重新获取**（需联网，且教师已上传题库）。
3. 点击 **开始任务** 答题。
4. 系统从题库中**随机抽题**（抽题数量由题库设定）。
5. 交卷后分数自动上传；**90 分及以上**视为通过。

### 3.4 上传作业

1. 在对应任务节点点击 **开始任务**。
2. 在文件对话框中选择本地文件。
3. 等待进度条完成；界面会显示提交时间与文件名。
4. 须**先成功登录**；上传失败时请检查网络与服务器地址。

### 3.5 电火花虚拟加工

完成前期任务后，可进入仿真场景进行电火花加工操作（装夹、对刀、加工等）。仿真场景中另有「整理仪器」小游戏，需按正确顺序摆放仪器。

## 四、教师端操作指南

教师登录后进入 **Tscene**，可管理学生、上传教学资源、下载学生作业。

### 4.1 菜单功能

| 菜单 | 功能 |
|------|------|
| **D** | 下载：选中学生的信息表、分工、模型、工艺、加工记录、检验、总结等（打包 zip） |
| **U** | 上传：实验指导书、报告模板、考试题库、PDF、视频链接等公共资源 |
| **S** | 返回主菜单 / 退出 |
| **O** | 用户列表：载入、插入、删除、全选等 |

### 4.2 上传公共资源（U 菜单）

| 上传项 | 学生端用途 |
|--------|------------|
| 实验指导书 | 公共资料 |
| 实验报告模板 | 公共资料 |
| 实验安全考试 | 安全操作考试题库 |
| 职工素养测试 | 职业工程素养测试题库 |
| 实验目的内容 | 任务 0 下载的 zip |
| m2、m3 | 设备仪器 PDF、实验指导书 PDF |
| videos | 视频链接文本（可编辑后提交） |

**每学期开课前建议**：上传 m2、m3 PDF、实验目的内容 zip，以及两份考试题库。

### 4.3 批量下载学生作业（D 菜单）

1. 在用户列表中勾选学生（可多选）。
2. D 菜单选择作业类型。
3. 选择本地保存目录，系统下载 zip 压缩包。

### 4.4 学生账号管理

- 按专业、班级、完成状态筛选学生。
- O 菜单可单个添加、批量导入（Excel）、删除学生。
- 点击学生可查看成绩、各文件提交时间、阅读时长等。

## 五、题库编制说明（教师）

题库须符合系统格式，否则考试无法正常使用。示范文件见工程目录 `教师示范/`。

### 5.1 文件格式

- 推荐：**Excel 2007+（.xlsx）**
- 备选：**.csv**（UTF-8 带 BOM）
- 也支持 **.xls**

### 5.2 表格结构

```
第 1 行  A 列可写「题目数」  B 列 = 每次考试随机抽取的题目数量（整数）
第 2 行  表头：序号、题型、题目、答案、选项A、选项B、选项C、选项D
第 3 行起  具体题目
```

### 5.3 题型与答案

| 题型 | 「题型」列 | 「答案」列 |
|------|------------|------------|
| 选择题 | `选择` | `A`、`B`、`C` 或 `D` |
| 填空题 | `填空` | 标准答案（不区分大小写，自动去空格） |

- 选择题选项写在「选项A」～「选项D」列；考试时选项顺序会随机打乱。
- **及格线：得分 ≥ 90**（按正确题数比例计分，满分 100）。

### 5.4 名称对应

| 课程中的考试 | 教师上传时选择的名称 |
|--------------|----------------------|
| 安全操作考试 | 实验安全考试 |
| 职业工程素养测试 | 职工素养测试 |

上传后，学生在考试任务中点击「重新获取」即可下载最新题库。

## 六、常见问题（用户）

### 6.1 登录失败

- 确认服务器地址完整、正确（含 `http://`、IP、`:8899` 及 `/api/values`）。
- 在浏览器打开 `…/health` 检查服务是否正常。
- 核对用户名、密码；联系教师确认账号是否已创建。
- 若曾填错地址：在登录页修改后点确认即可覆盖保存。

### 6.2 上传失败

- 确认已成功登录。
- 检查校园网 / VPN 是否连通服务器。
- 单个文件较大时请耐心等待；持续失败请联系管理员。

### 6.3 考试无法获取题库

- 确认教师已在 U 菜单上传对应题库。
- 点击「重新获取」；仍失败则检查网络或联系教师。

### 6.4 PDF 无法打开

- 确认教师已上传 m2、m3 文件。
- 确认登录页服务器地址与 API 使用同一 IP 和端口（PDF 地址由该地址自动推导）。

### 6.5 成绩未更新

- 交卷后应出现「分数已更新」提示；若无提示，检查网络后重新登录查看。
- 仍异常请联系教师从管理端查看该生 score1 / score2 字段。
