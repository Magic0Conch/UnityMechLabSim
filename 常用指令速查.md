# UnityMechLabSim 常用指令速查

> 适用当前环境：Windows 本机开发 + Ubuntu 服务器部署（`ubuntu@49.233.160.65`）。

## 1) 本地后端（Windows）

### 启动（开发模式）
```powershell
cd "e:\work\Unity\UnityMechLabSim-main\server"
if (-not (Test-Path .\.venv\Scripts\uvicorn.exe)) {
  python -m venv .venv
  .\.venv\Scripts\pip install -r requirements.txt
}
.\.venv\Scripts\uvicorn app.main:app --host 127.0.0.1 --port 8899 --reload
```

### 健康检查
```powershell
python -c "import urllib.request;print(urllib.request.urlopen('http://127.0.0.1:8899/health',timeout=5).read().decode())"
```

### 重启本地服务（手动）
```powershell
netstat -ano | findstr :8899
taskkill /PID <PID> /F
cd "e:\work\Unity\UnityMechLabSim-main\server"
.\.venv\Scripts\uvicorn app.main:app --host 127.0.0.1 --port 8899 --reload
```

## 3) 服务器服务管理（systemd）

### 重启/查看状态
```bash
sudo systemctl restart mechlab
sudo systemctl status mechlab --no-pager
```

### 查看日志
```bash
sudo journalctl -u mechlab -n 50 --no-pager
sudo journalctl -u mechlab -f
```

### 确认服务实际工作目录
```bash
sudo systemctl cat mechlab | grep -E "WorkingDirectory|ExecStart"
```

### 服务器健康检查
```bash
curl -s http://127.0.0.1:8899/health
```

## 4) 数据库查询（SQLite）

数据库路径（当前部署）：
`/home/ubuntu/mechlab-server/server/data/mechlab.db`

### 进入 sqlite
```bash
sqlite3 /home/ubuntu/mechlab-server/server/data/mechlab.db
```

### 常用 SQL
```sql
.tables
.schema users

SELECT username, role, password
FROM users
WHERE username='yigengli@ysu.edu.cn';

SELECT username, role, name
FROM users
WHERE role='teacher'
ORDER BY username;

SELECT role, COUNT(*) FROM users GROUP BY role;
```

### 一条命令直接查（不进 sqlite）
```bash
sqlite3 /home/ubuntu/mechlab-server/server/data/mechlab.db \
"SELECT username, role, password FROM users WHERE username='yigengli@ysu.edu.cn';"
```

### 强制把账号改成教师
```bash
sqlite3 /home/ubuntu/mechlab-server/server/data/mechlab.db \
"UPDATE users SET role='teacher', password='ysu138352' WHERE username='yigengli@ysu.edu.cn';"
```

## 5) 接口联调（快速判断后端返回）

### 测试登录返回值（服务器本机）
```bash
curl -s -X POST "http://127.0.0.1:8899/api/values" \
  -F "requesttype=LOGIN" \
  -F "username=yigengli@ysu.edu.cn" \
  -F "password=ysu138352"
```

返回通常是以下之一：
- `TEACHERSUCCESS`
- `STUDENTSUCCESS`
- `FAILED`

### 测试学生信息接口
```bash
curl -s -X POST "http://127.0.0.1:8899/api/values" \
  -F "requesttype=GEIINFO" \
  -F "username=student1"
```

## 7) 常见故障快查

- 重启 `mechlab` 没输出：通常正常，再看 `status`。
- 插入教师后仍进学生端：先查数据库 `role` 是否为 `teacher`。
- 改了代码不生效：检查是否传到了服务真实目录、是否重启了正确服务。
- 本地端口冲突：`netstat -ano | findstr :8899` 找 PID 后 `taskkill`。
